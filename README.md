DESCRIPCIÓN DEL PROYECTO

El Sistema de Gestión de Inventario de Biblioteca es una aplicación de escritorio desarrollada en
Java SE (Swing) que permite administrar los recursos de una biblioteca.
El sistema ofrece un entorno amigable para gestionar usuarios, libros y préstamos, garantizando
la persistencia de datos mediante una base de datos SQLite.
El propósito del proyecto es optimizar el control del inventario en bibliotecas pequeñas o
medianas que no disponen de conexión a internet o de sistemas automatizados, mejorando la
organización y eficiencia del encargado.
